# Margo SDK

This repository provides a Go SDK for building Margo-compliant WOS, Devices using clients, CLIs, and server helpers.  
It is intended for developers implementing or integrating with the Margo specification.

## Overview

- **Purpose:** Offer reusable models, API clients, authentication, and package management helpers for Margo Spec.
- **Not a server:** This SDK does not implement a server, but provides building blocks so you can use them, if needed.
- **Docs:** See the [`docs/`](./docs) directory for detailed guides.

## Quick Start

```sh
git clone https://github.com/margo/dev-repo.git
cd dev-repo
# See docs/navigating-codebase.md for structure
```
